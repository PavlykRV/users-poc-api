## users-poc-api

# Description

Proof of Concept for User management service. Basically provide CRUD for users.

# Requirements

- Node 14+ (v14.15.3)
- npm 6+ (6.14.9)
- MongoDB 5+ (v5.0.3)
- Git

## Common setup

```bash
git clone https://github.com/PavlykRV/users-poc-api.git
cd users-poc-api
```

```bash
npm install
```

## Run app

To start the express server, run the following

```bash
node server.js
```
